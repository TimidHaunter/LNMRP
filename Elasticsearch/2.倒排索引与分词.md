## Analyzer

分词器，分析器

Lucene：负责`倒排索引`物理构建

Analyzer：负责在建立倒排索引前和搜索前对文本进行`词法和语法`处理



## 倒排索引

正排索引：通过文档ID关联文档内容
倒排索引：通过关键单词关联ID



### 简单倒排索引

![简单的倒排例子](G:\data\LNMRP\$Image\ElasticSearch\es_invert_index.png)



### 复杂倒排索引

- 会包含文档编号

|   单词   | 文档ID（包含文档编号） |
| :------: | :--------------------: |
| 搜索引擎 |     【1,1】【3,1】     |



- 包含单词出现位置

|   单词   | 文档ID（包含文档编号、单词位置） |
| :------: | :------------------------------: |
| 搜索引擎 |      【1,1,<6>】【3,1,<0>】      |

## 应用场景

### 索引时使用

插入编辑文档时，对文档分析

```json
##大写变小写，去掉了标点符号；分词
POST _analyze
{
  "analyzer": "standard",
  "text": "My First Day at Senior High!"
}

##和standard区别不大
POST _analyze
{
  "analyzer": "simple",
  "text": "My First Day at Senior High!"
}

##根据空格切分英文单词，不变换大小写，去掉了标点符号
POST _analyze
{
  "analyzer": "whitespace",
  "text": "My First Day at Senior High!"
}

POST _analyze
{
  "analyzer": "simple",
  "text": "My First Day at Senior High!"
}
```



## 设置Analyzer

在创建索引的时候指定分词类型

### setting

```json
##在setting参数中指定在索引的所有文本字段中使用standard进行索引构建
PUT /case1
{
  "settings": {
    "analysis": {
      "analyzer": {
        "default":{
          "type":"standard"
        }
      }
    }
  },
  "mappings": {
    "properties": {
      "name":{
        "type":"text"
      },
      "address":{
        "type":"text"
      }
    }
  }
}
```



### mapping

```json
##在mapping参数中指定在索引的name字段中使用 whitespace，address字段使用 simple 进行索引构建
PUT /case2
{
  "mappings": {
    "properties": {
      "name":{
        "type":"text",
        "analyzer":"whitespace"
      },
      "address":{
        "type":"text",
        "analyzer":"simple"
      }
    }
  }  
}
```



### 搜索

```json
##默认analyzer和search_analyzer一致
PUT /case3
{
  "mappings": {
    "properties": {
      "name":{
        "type":"text",
        "analyzer":"whitespace",           //索引
        "search_analyzer":"whitespace"     //搜索
      }
    }
  }  
}
```



### 自定义

系统不满足时，使用自定义

```json
##自定义分词器
PUT /case4
{
  "settings":{
    "analysis": {
      "analyzer": {
        "custom_analyzer":{        //自定义的ayalyzer的名称，随便起的
          "type":"custom",         //自由搭配
          "tokenizer":"standard",  //tokenizer：分词方式
          "filter":["lowercase"]   //filter：过滤方式
        }
      }
    }
  }
}

##测试自定义分析，转了小写
POST /case4/_analyze
{
  "text":"My First Day at Senior High!"
}
```



## 中文分词器

IK，基于Java语言开发

docker下安装IK分词器

安装在`elasticsearch/plugins/analysis-ik`



### IK_smart

粗粒度切分词



### IK_max_word

 细粒度切分词

```json
##IK_smart分词器
POST _analyze
{
  "analyzer":"ik_smart",
  "text":"我爱北京天安门，天安门上太阳升"
}

##IK_max_word分词器
POST _analyze
{
  "analyzer":"ik_max_word",
  "text":"我爱北京天安门，天安门上太阳升"
}
```

## IK同义词

### 建立索引时

指定同义词并构建同义词倒排索引

```json
##定义索引
PUT /city
{
  "settings": {
    "analysis": {
      "filter": { 
        "ik_synonyms_filter":{                  //自定义filter
          "type":"synonym",
          "synonyms":[
            "北京,首都,京城,北平,燕京,帝都",     //英文逗号[,]
            "上海,魔都,沪上",
            "广州,妖都,羊城",
            "成都,蓉城,基都"
          ]
        }
      },
      "analyzer": {
        "ik_analyzer_synonyms":{                //自定义analyzer
          "tokenizer":"ik_smart",
          "filter":[
            "lowercase",                        //内置filter
            "ik_synonyms_filter"                //自定义filter
          ]
        }
      }
    }
  },
  "mappings": {
    "properties": {
      "title":{
        "type":"text",
        "analyzer": "ik_analyzer_synonyms"      //自定义analyzer
      }
    }
  }
}

##插入数据
POST /city/_doc/001
{
  "title":"北京"
}
POST /city/_doc/002
{
  "title":"上海"
}
POST /city/_doc/003
{
  "title":"广州"
}
POST /city/_doc/004
{
  "title":"成都"
}
```

### 查询时

```json
##添加索引，查询时，ik分词
PUT /country
{
  "settings": {
    "analysis": {
      "filter": {
        "country_synonyms_filter":{
          "type":"synonym_graph",                     //
          "updateable":true,                          //动态更新ES同义词库，没有必要重启服务
          "synonyms_path":"analysis-ik/synonyms.txt"  //同义词库路径
        }
      },
      "analyzer": {
        "ik_synonyms_search_analyzer":{	
          "tokenizer":"ik_max_word",
          "filter":[
            "lowercase",                              //内置过滤器
            "country_synonyms_filter"                 //自定义过滤器
          ]
        }
      }
    }
  },
  "mappings": {
    "properties": {
      "title":{
        "type":"text",
        "analyzer":"ik_max_word",
        "search_analyzer": "ik_synonyms_search_analyzer" 
      }
    }
  }
}

##查看分词效果
POST /country/_analyze
{
  "analyzer":"ik_synonyms_search_analyzer",
  "text":"西安是中国的十三朝古都"
}

##插入数据
POST /country/_doc/1

}

POST /country/_doc/2
{

}

##动态更新同义词库
中国 CHINA 中华人民共和国
民国 中华民国 
```

## note

明确字段是否需要分词，不需要分词的字段就将 type 设置为 keyword，可以节省空间和提高写性能。
